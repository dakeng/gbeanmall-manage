# GBeanMall-manage

游戏金豆兑换平台后台管理系统

- 使用react-create-app脚手架创建
- 使用Ant Design的UI库
- Express + MongoDB (Mongoose)

## 遇到的问题

- [ ] 解决antd按需加载找不到模块的问题
- [ ] 表单弹窗关闭时取消请求
- [ ] 熟悉promise和fetch

  ​
## 运行项目

``` git clone https://github.com/dakeng/gbeanmall-manage.git ```
``` npm install ```
``` npm start ```
``` http://localhost:3000/admin ```
